# Online Food Ordering System

## Description
A simple food ordering system for restaurants that allows users to browse food items, add them to a cart, and checkout. Admins can manage food items and orders. This project is built using **HTML, CSS, and Java.

## Features
- Display food items dynamically on a web page
- Add items to the cart and view total price
- Checkout process to confirm orders
- Admin dashboard to add, remove, or update food items (optional)
- Orders are stored in memory using Java collections

## Tech Stack
- **Frontend:** HTML, CSS
- **Backend:** Java (Core Java / Servlets for advanced version)
- **Database:** Simulated using Java ArrayLists
